# Recommendation System Using Java

## Description

This project is a simple Recommendation System developed in Java. The program recommends courses based on the user's preferred technology category. It demonstrates the basic concept of personalized recommendations using conditional statements and user input.

## Features

* Takes user input through the console
* Recommends courses based on selected interests
* Supports multiple categories:

  * Java
  * Python
  * Web Development
* Simple and beginner-friendly implementation
* Demonstrates decision-making using if-else statements

## Technologies Used

* Java
* Scanner Class
* Conditional Statements

## How It Works

1. The user enters a preferred category.
2. The system checks the selected category.
3. Based on the category, relevant courses are recommended.
4. If the category is not available, a default message is displayed.
